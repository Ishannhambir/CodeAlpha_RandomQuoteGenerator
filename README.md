# CodeAlpha Random Quote Generator

This is my implementation of Task 2 from the CodeAlpha App Development Internship. It's a simple web-based app that generates random quotes, built with HTML, CSS, and JavaScript.

## Features
- Displays a random quote with text and author name on load.
- Click the "New Quote" button to get a different quote each time.
- Uses a built-in list of inspirational quotes (offline-friendly).
- Clean, minimal, and colorful UI with gradients and hover effects for a better user experience.

## How to Run
1. Clone or download the repo.
2. Open `index.html` in any web browser (e.g., Chrome).
3. Use the app directly—no internet or installation needed!

## Technologies Used
- HTML5 for structure.
- CSS3 for styling (with gradients and animations).
- JavaScript for random quote generation.
